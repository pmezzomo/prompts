# Autoconsistência (Self-Consistency)

**Definição:** Self-Consistency Prompting é a técnica de pedir para a IA gerar várias respostas diferentes para o mesmo prompt e depois analisar qual delas é a melhor (ou você mesmo faz essa escolha). O objetivo é obter a resposta mais lógica, clara e eficaz a partir da comparação de ideias.

---

## 🕒 Quando usar

- Quando há múltiplas formas válidas de resolver uma questão.
- Para gerar variações de ideias, estratégias, argumentos ou soluções.
- Em decisões críticas, brainstorms ou construção de conteúdo estratégico.

---

## 🧩 Template

```
Responda essa pergunta de 3 formas diferentes, usando raciocínios distintos.
Depois, analise e diga qual é a melhor resposta e por quê.
```

---

## 💼 Exemplo

```
Como melhorar o engajamento de clientes em um e-commerce?

Responda com 3 abordagens diferentes (marketing, UX e pós-venda).

Depois, diga qual estratégia é mais indicada para empresas com orçamento limitado e justifique.
```

---

## 🎯 Vantagens

- Estimula pensamento crítico e diversidade de soluções.
- Reduz risco de aceitar uma resposta superficial ou precipitada.
- Ideal para decisões, comparações e criatividade estratégica.
