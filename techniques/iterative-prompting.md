# Prompt Iterativo (Iterative Prompting)

**Definição:** Iterative Prompting é a técnica de ajustar a resposta da IA em ciclos curtos, com base em feedbacks progressivos, até atingir o resultado desejado. O objetivo é melhorar uma resposta aos poucos, sem recomeçar do zero, mantendo o contexto da conversa.

---

## 🕒 Quando usar

- Quando a resposta ficou quase boa, mas precisa de ajustes.
- Ao criar conteúdos que exigem revisão de tom, estrutura ou exemplos.
- Em tarefas que envolvem aprimoramento contínuo ou prototipagem.

---

## 🧩 Template

```
Reescreva esse texto de forma mais objetiva.

Adicione exemplos práticos no contexto de [área].

Reduza para até [X] palavras, mantendo o ponto principal.

Agora adapte esse conteúdo para o público [perfil].
```

---

## 💼 Exemplo

```
Liste os principais desafios enfrentados por uma empresa.
```

---

## 🎯 Vantagens

- Permite ajustar sem perder o histórico da conversa.
- Ideal para conteúdos complexos que exigem refinamento.
- Excelente para prototipar textos, roteiros, planos ou comunicações.
