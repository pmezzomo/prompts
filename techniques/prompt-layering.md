# Prompt Layering (Empilhamento de Prompts)

**Definição:** Prompt Layering (ou empilhamento de prompts) é a técnica de quebrar uma tarefa complexa em várias etapas sequenciais de prompts. Cada etapa constrói sobre a anterior, como camadas. O objetivo é construir respostas complexas com controle, clareza e profundidade, etapa por etapa.

---

## 🕒 Quando usar

- Em projetos com várias fases ou camadas de conteúdo.
- Para criar planos, apresentações, cursos, campanhas, estratégias.
- Quando a tarefa é complexa e precisa ser dividida para ficar mais precisa.

---

## 🧩 Template

```
Etapa 1: Estruture os blocos principais sobre [tema].

Etapa 2: Desenvolva os tópicos definidos na Etapa 1 com explicações claras.

Etapa 3: Agora reescreva todo o conteúdo no formato [X] (artigo, slide, roteiro, etc.).
```

---

## 💼 Exemplo

```
Etapa 1: Liste como usar Inteligência Artificial voltado para profissionais de marketing digital.

Etapa 2: Detalhe os tópicos da etapa anterior com foco em aplicações práticas no planejamento, execução e análise de campanhas.

Etapa 3: Organize tudo no formato para enviar por e-mail.
```

---

## 🎯 Vantagens

- Permite construir conteúdos complexos com clareza.
- Mantém a coerência entre partes e etapas.
- Dá mais controle e evita que a IA se perca ou pule passos importantes.
