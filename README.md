# Prompt Engineering

Personal knowledge base on prompt engineering techniques for working with AI models.

## Structure

```
prompts/
├── prompt-engineering.md   # Main file — full table with all 14 techniques
└── techniques/             # One file per technique with detailed content
```

## What's inside

**`prompt-engineering.md`** — overview table with all techniques, their purpose, and links to individual files.

**`techniques/`** — each file covers:
- Definition
- When to use
- Prompt template
- Practical example
- Advantages

## Techniques covered

| # | Technique | Focus |
|---|-----------|-------|
| 1 | Zero-Shot | Direct instruction, no examples |
| 2 | Few-Shot | Guide response with examples |
| 3 | Chain of Thought | Step-by-step reasoning |
| 4 | Follow-Up | Refine or expand previous responses |
| 5 | Iterative Prompting | Improve responses in short cycles |
| 6 | Two-Step Prompt | Structure first, then generate |
| 7 | Prompt Layering | Build complex content in layers |
| 8 | Self-Consistency | Compare multiple answers, pick the best |
| 9 | Memory Anchoring | Keep tone and context consistent |
| 10 | Meta-Prompt | Create or improve other prompts |
| 11 | Question Refinement | Make vague questions more specific |
| 12 | Cognitive Verification | Validate the AI's own reasoning |
| 13 | User Persona | Adapt response to a specific audience |
| 14 | Role Prompting | Assign a professional role to the AI |
